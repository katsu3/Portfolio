		- 開発は進むが、全体を見渡して推進できる人材がいない

		- スタートアップや新規事業で、開発体制をゼロから構築したい
- 開発プロセスを改善したい
		- 属人化や非効率が課題で、仕組み化・標準化を進めたい
- AI関連プロジェクトを始めたい

- 外部の視点が欲しい
		- 社内だけでは解決が難しい課題に、第三者の知見を取り入れたい

## 概要
PM/EMとして、ビジネス要求と技術的実現性の両面から開発チームをリード。  
フロントエンド・バックエンド・ネイティブアプリ・生成AI領域に至るまで、幅広いプロジェクトでマネジメントおよび技術推進を担当。  
特に、複数チームを跨いだ大規模プロジェクト推進や、技術組織の立ち上げ・仕組み化に強みを持つ。

<br>


## 得意領域
- **プロジェクトマネジメント （PM / PdM / PMO）**
	- 35名規模の組織運営・リソース最適化
	- 複数PJの並行管理とステークホルダ調整
	- ビジネス要求の技術転換・優先度設計

- **エンジニアリングマネージャー（EM）**
	- 組織設計・人材育成・技術カルチャー醸成
	- 技術基盤整備と品質改善施策の推進
	- AI・Web・アプリ開発における横断的リーダーシップ
